# Web Scraping Project using Python

## Project Overview
This project demonstrates how to perform web scraping using Python. The program extracts real quotes, author names, and tags from a public website and stores the data in an Excel file. This project shows how to collect and organize data from web pages for analysis.

---

## Objective
The main objective of this project is to:
- Extract real quotes and author names from a website
- Learn how to use Python for web scraping
- Create a structured dataset
- Save the extracted data in Excel format

---

## Tools and Technologies Used
- Python
- Requests library
- BeautifulSoup library
- Pandas library
- Excel (openpyxl)
- VS Code
- GitHub

---

## How the Project Works

1. The program sends a request to the website using the Requests library.
2. The HTML content of the webpage is collected.
3. BeautifulSoup parses the HTML and finds quotes and author information.
4. The extracted data is stored in lists.
5. Pandas converts the data into a table format (DataFrame).
6. The data is saved into an Excel file.

---

## Output

The program generates an Excel file containing:

- Quote
- Author
- Tags
